Static position. These elements are considered to be "not positioned".
Relative position. It operates the same as "static", unless you give it extra values.
Giving values to the top, right, bottom, and left will move the element. These values are relative to its default position.
Absolute position. Behaves like "relative", except that it is relative to its parent element. If parent is not "relative", it will jump to the next parent until it finds something "relative" to relate to.
Fixed position. It will stay visible in the screen, even if it is scrolled through.
